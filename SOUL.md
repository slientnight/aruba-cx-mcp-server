# SOUL — Skills & Capabilities

## Aruba CX Switching Skills (1)

### aruba-cx-ops

Manage Aruba CX switches via the AOS-CX REST API. Capabilities include: system info and health monitoring, interface listing and configuration, VLAN management (create/delete), configuration management (running/startup config, write memory, checkpoints), routing and ARP table inspection, LLDP neighbor discovery, MAC address table lookup, optics and DOM diagnostics with threshold-based health assessment, ISSU (in-service software upgrade) with readiness checks and rollback, firmware upload/download and status tracking, VSF stack topology, and multi-switch target management. Write operations require a ServiceNow Change Request number when ITSM gate is enabled.
